🔐 Secure SDLC Demonstration Web Application

This project demonstrates implementation of Secure Software Development Lifecycle (SSDLC) practices using a Flask web application.

🛡 Security Features

Password Hashing (PBKDF2)
HTTPS Encryption
Input Validation
SQL Injection Prevention
Security Logging
Brute Force Login Protection
Secure HTTP Headers

🔐 Privacy by Design

The application collects minimal personal data and secures credentials using cryptographic hashing while ensuring encrypted communication.

🏗 Architecture

User → HTTPS Request → Flask Application → Input Validation → Password Hashing → Database → Logging

🚀 Technologies Used

  • Python
  • Flask
  • SQLite
  • OpenSSL
  • GitHub
